 A Simple Web Calculator 

A responsive calculator web application built using HTML, CSS, and JavaScript. 
This project demonstrates core front-end development concepts including DOM manipulation and event handling.


## Features

- Basic arithmetic operations (+, −, ×, ÷)
- Percentage calculation (%)
- Square (x²)
- Square root (√x)
- Cube root (∛x)
- Power function (xʸ)
- Delete (DEL) and Clear (C)
- Decimal support
- Real-time display updates
- Keyboard input support
- Error handling for invalid expressions


## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)


## How to Run

1. Clone the repository
2. Open `CALC.html` in your browser
3. Use the calculator
